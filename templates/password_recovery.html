Password Recovery Guide

How It Works

  1. Enter your email: Provide the email address associated with your account
  2. Receive verification code: Check your email for a 6-digit verification code
  3. Verify your identity: Enter the code to confirm you own the account
  4. Create new password: Set a strong, secure password following our requirements

Security Tips

  • Never share your verification code with anyone
  • Codes expire after 15 minutes for security
  • Use a unique password that you don't use elsewhere
  • Consider using a password manager for better security
  • Enable two-factor authentication after resetting your password

Troubleshooting

  • Didn't receive the email? Check your spam folder and wait up to 5 minutes
  • Code expired? Request a new verification code
  • Wrong email address? Make sure you're using the correct email associated with your account
  • Still having issues? Contact our Account Assistance team

Alternative Recovery Methods

Phone Number Recovery

If you have a verified phone number on your account, you can receive a verification code via SMS.

Security Questions

If you've set up security questions, you can use them to verify your identity.

Contact Support

If none of the above methods work, our support team can help you recover your account.
